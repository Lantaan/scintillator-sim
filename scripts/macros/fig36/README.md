This folder contains macros for reproducing a Figure 3.6-style absorption-efficiency
comparison from the Ovsiannikova thesis:

- 51 x 51 x 10 mm CeBr3
- no reflector vs reflector A
- energies: 20, 50, 100, 200, 500, 1000, 4000 keV
- 10000 primaries per run

Use `scripts/run_fig36_repro.sh` to run all cases and archive outputs safely.
